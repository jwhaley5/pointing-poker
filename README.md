# Pointing Poker

A lightweight real-time planning poker app for agile teams.

Built with React, TypeScript, SST, API Gateway WebSockets, and DynamoDB.

## Features

- Real-time voting and reveal flow
- Member and observer roles
- Multiple rounds with custom titles
- Vote history
- Mobile-friendly UI

## Local Development

```bash
npm install
cd shared && npm install && cd ..
cd frontend && npm install && cd ..
npm run dev
```
